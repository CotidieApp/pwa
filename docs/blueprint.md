# **App Name**: SoulSpace

## Core Features:

- Prayer Entry: Users can add their prayers, recording the date, time, and specific content.
- Category Assignment: Assign prayers to pre-defined categories such as 'Gratitude', 'Supplication', or create custom categories.
- Prayer Browser: Browse prayers by category for focused reflection and review.
- Prayer Search: A simple search function allows users to quickly locate specific prayers by keywords.

## Style Guidelines:

- Primary color: Soft blue (#A0BBD9) to evoke a sense of calmness and reflection.
- Background color: Very light blue (#F0F4F8), almost white, for a clean and serene backdrop.
- Accent color: Pale gold (#D4AC0D) to highlight key interactive elements such as buttons or selected categories.
- Body and headline font: 'Literata' serif, to offer a literary, slightly formal feel.
- Simple, elegant icons to represent each prayer category (e.g., a heart for Gratitude, clasped hands for Supplication).
- Clean, minimalist layout to reduce visual clutter and encourage focus. Ample spacing between elements.
- Gentle fade-in animations for newly added prayers or when transitioning between categories.
